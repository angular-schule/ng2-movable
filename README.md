# ng2-movable

Demo: [https://spali.github.io/ng2-movable/](https://spali.github.io/ng2-movable/)

## How to install

```bash
$ npm install ng2-movable --save
```

## How to demo

```bash
$ git clone https://github.com/spali/ng2-movable.git
$ cd ng2-movable
$ npm install
$ npm start
```
